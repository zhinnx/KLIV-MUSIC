# KLIV - Music Streaming

Mobile-first music streaming app built with Next.js, Tailwind, Zustand, Supabase Auth.

## Features implemented (Step-by-step):
- ✅ Onboarding (splash + 3 slides + Login Google / GitHub)
- ✅ Homepage (Beranda)
- ✅ Search
- ✅ Library (Pustaka)
- ✅ Playlist detail (with lock/reorder + bottom sheet)
- ✅ Profile (logged in + logged out)
- ✅ Settings
- ✅ Artist profile
- ✅ Mini Player (sticky)
- ✅ Fullscreen Music Player
- ✅ Real Supabase OAuth login (Google + GitHub)
- ✅ Player state management (Zustand + mock stream)
- ✅ Mobile-first + Stitch design system (lime accent, dark #071014)

## Tech
- Next.js 16 + Tailwind 4
- Supabase Auth (OAuth)
- Zustand (Player)
- Framer Motion (optional)
- Mock data + api-velomusic fallback for streaming

## Run
```bash
npm install
npm run dev
```

Open http://localhost:3000

Login via onboarding or profile.
